My Favorite Data Science Computing Stack

My favorite – if rather expensive – setup for large-scale Data Science and Machine Learning projects involves:

  • An AWS Elastic MapReduce server cluster with:

  • the Master node being a powerful G2.2xLarge AWS EC2 instance with a high-end NVIDIA GRID Graphics Processing Unit (GPU);

  • Apache Hadoop for distributed storage;

  • Apache Spark for distributed parallel computing; Spark ships with:

    • Spark SQL for HiveQL queries on distributed data frames / data tables; and
    • Spark ML for Machine Learning algorithms on distributed data frames / data tables;

  • Python v2.7, which is integrated with Spark through the PySpark API;

  • The SciPy scientific computing stack with packages:

    • NumPy for linear algebra;
    • Pandas for data frames;
    • SciPy for advanced functions and algorithms;
    • SymPy for symbolic math;
    • MatPlotLib for visualization; and
    • iPython / Jupyter, an interactive Python environment;

  • NVIDIA's CUDA toolkit and DeepLearning.net's Theano package for compiled parallel computing through the GPU;

  • Basic Linear Algebra Subroutines ("BLAS") libraries BLAS, LAPACK, ATLAS, OpenBLAS, and NVIDIA's cuBLAS and NVBLAS – here sorted in the general order of increasing sophistication and performance;

  • General Machine Learning packages and "SciKits" such as SciKit-Learn, MLpy, Orange, StatsModels, SciKit-Image and the Natural Language Toolkit (NLtk);

  • Deep Learning packages – an overwhelming plethora of them indeed! – such as Blocks, Brainstorm, Caffe, Chainer, CUDA-ConvNet, DeepDish, DeepDist, Deepy, DeepPy, FANN, FFnet, Hebel, Keras, Lasagne, Minerva, Mozi, NervanaNEON, NeuralPy, NeuroLab, NLPnet, NN, NoLearn, OpenANN, OpenDeep, OpenDL, PDNN, PyBrain, PyBrain2, PyDeepLearning, PyDNN, PyLearn2, PythonBrain, SciKit-NeuralNetwork, SKflow, TensorFlow, Theanets and Veles;

  • Graph and Network analysis toolkits such as GraphViz, Graph-Tool, iGraph, NetworkX and SNAPpy;

  • Geospatial analysis toolkits such as Basemap; and

  • Advanced visualization packages Bokeh, GGplot and Plotly and Seaborn.

I have coded scripts for bidding for an AWS EMR cluster at spot EC2 prices and installing the aforementioned software on the cluster. Please feel free to leverage these scripts for your own benefit. :)
